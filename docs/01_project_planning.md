# 📘 Project Planning & Management

## 1. Project Title

Spirit of Ankh is a Smart Tourism Crowd Management System in Egypt  

---

## 2. Project Overview

This project aims to develop a smart system that monitors and manages tourist crowd levels in real-time across Egyptian tourist attractions.

The system uses data pipelines and analytics to provide insights that help in decision-making and improving visitor experience.

---

## 3. Problem Statement

Tourist attractions in Egypt face high congestion, especially after the opening of new major attractions like the Grand Egyptian Museum.

There is no real-time system to:

* Monitor crowd levels
* Detect abnormal increases
* Predict future congestion

---

## 4. Objectives

* Monitor real-time crowd density
* Detect abnormal spikes in visitors
* Analyze crowd behavior over time
* Provide insights through dashboards

---

## 5. Project Scope

### In Scope:

* Real-time crowd estimation
* Crowd Persistence Index (CPI)
* Anomaly detection (alerts)
* Dashboard visualization

### Out of Scope:

* Mobile applications
* Full IoT sensor integration
* Real-time ticketing systems

---

## 6. Deliverables

* Data Pipeline (Simulation + Processing)
* KPI Calculations (CPI, Alerts)
* Dashboard (Power BI)
* UI Interface
* Documentation

---

## 7. Team Roles

* Data Engineer: Build data pipeline
* Data Analyst: Dashboard & insights
